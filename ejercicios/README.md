# 🏋️ Ejercicios Prácticos de Puppeteer

Aquí encontrarás ejercicios para practicar lo que aprendiste en los ejemplos básicos.

## 📖 Cómo Usar Esta Carpeta

1. **Abre un ejercicio** (ej: `ejercicio-01.js`)
2. **Lee las instrucciones** en los comentarios
3. **Escribe tu código** en la sección marcada
4. **Ejecuta tu solución**: `node ejercicios/ejercicio-01.js`
5. **Si te atascas**, revisa la solución en `soluciones/ejercicio-XX-solucion.js`

## 💡 Consejos

- ✅ **Intenta resolverlos sin ver la solución primero**
- 🔍 **Usa console.log()** para debuggear tu código

- 🐢 **Usa slowMo** para ver qué hace tu código en cámara lenta
